Suppress warnings

Specify the warnings to supress by setting these in the suppressWarnings configuration element. This can be done using a comma separated list or as nested suppressWarning configuration elements.

<project>
  ...
  <build>
    <plugins>
      ...
      <plugin>
        <groupId>com.github.robtimus</groupId>
        <artifactId>i18n-maven-plugin</artifactId>
        <version>3.0</version>
        <executions>
          <execution>
            <goals>
              <goal>generate</goal>
            </goals>
            <configuration>
              <bundleName>com.foo.bar.i18n</bundleName>
              <suppressWarnings>javadoc,nls</suppressWarnings>
            </configuration>
          </execution>
        </executions>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>
